About Curabit

A social venture that aims to improve mental health treatment in India through virtual reality (VR).

Founded by Aman Sariya and me, Curabit aims to empower mental health professionals with tools that help them identify and treat disorders more effectively, accurately, conveniently and safely.

The main idea is to use VR as a tool to introduce audio and visual anchors to create a sense of presence for the patient. It allows our VR-based therapy to be more stimulatory than traditional therapeutic techniques. It is also cost-effective and allows for bio-monitoring. The working of our VR system can be highlighted using an use-case scenario.

UnityVR Application Development

A basic app that plays 360 videos and controls their playback through webhook communication.

Developed using the Unity framework, this application will be displaying various 360 videos that we have shot and stored locally on the headset. The application connects to an API endpoint, which send instructions on which video to play when, whether to seek forwards or backwards and when to stop playing.

I developed only a couple of base functions for this application (selection of 360 video for playback via API requests and responses as well as the media player for 360 videos). This project was further developed into a proper functioning application by my partner, Aman Sariya.
